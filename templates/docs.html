Quick Start

Get {{ brand.name }} running in under 3 minutes. No installer, no kernel drivers, no IT approval needed.

config.json

All settings are stored in AppData\Roaming\FMSecure\config.json. Changes take effect on next startup or via hot-reload from the dashboard.

config.json 
{
  "watch_folder": "C:\\Users\\YourUser\\Documents",
  "verify_interval": 60,          // seconds between periodic checks
  "webhook_url": "",              // Discord or Slack webhook URL
  "admin_email": "",              // SMTP alert recipient
  "vault_max_size_mb": 10,       // max file size to back up (PRO)
  "vault_allowed_exts": [        // extensions eligible for vault
    ".txt", ".json", ".py", ".html",
    ".js", ".css", ".md"
  ],
  "lolbin_whitelist": [          // trusted parent processes
    "Code.exe", "pycharm64.exe"
  ]
}

Mass Deployment

Silent deployment via environment variable — no user interaction required on target machines.

GPO / Registry — Silent Enrollment
# Method 1: Environment variable (GPO / MDM) FMSECURE_TENANT_KEY=fms-tenant-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx   # Method 2: Registry (HKLM for all users) reg add "HKLM\SYSTEM\CurrentControlSet\Control\Session Manager\Environment" \ /v FMSECURE_TENANT_KEY /t REG_SZ /d "fms-tenant-xxx..." /f   # Agent auto-enrolls on first launch — no wizard shown

Build Commands

Official build uses Nuitka for maximum performance. PyInstaller is also supported.

Nuitka — Recommended 
# Watchdog process
python -m nuitka --standalone --windows-disable-console \
  --output-dir=dist --output-filename=WinSysHost.exe \
  --lto=yes sys_watchdog.py

# Main agent
python -m nuitka --standalone --windows-disable-console \
  --enable-plugin=tk-inter --enable-plugin=anti-bloat \
  --output-dir=dist --output-filename=SecureFIM.exe \
  --windows-icon-from-ico=assets/icons/app_icon.ico \
  --include-data-dir=assets=assets \
  --include-package=customtkinter --include-package=pystray \
  --include-package=PIL --include-package=cryptography \
  --include-package=google --include-package=watchdog \
  --include-package=core --include-package=gui \
  --lto=yes run.py
