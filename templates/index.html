Core Capabilities

Multi-layer defense.
Single binary.

Every layer operates independently — a failure in one never compromises the others. No kernel driver signing. No IT department approval.

🔍
Real-Time File Integrity
Event-driven watchdog replaces polling. SHA-256 hashing with HMAC-signed records. Detects create, modify, delete, rename — even editor "atomic save" patterns. Debounce timer prevents false positives on large writes.
🛑
Ransomware Killswitch
Burst detection triggers on ≥5 file events in 10 seconds. OS-level lockdown via Windows icacls — strips Write/Delete for Everyone at NTFS kernel level. Honeypot tripwire detonates instantly.
🔒
AES-256 Auto-Heal Vault
Self-healing vault backs up critical files (under 10 MB) and restores them instantly on modification or deletion. AES-256 Fernet encryption at rest. Hardware-bound KEK via PBKDF2 200k iterations — stolen key files are permanently unreadable.
⚡
Process Attribution
Every file event records which process caused it. LOLBin detection identifies 25+ known-malicious Windows binaries (PowerShell, certutil, mshta, etc.). Trusted-parent whitelist suppresses false positives from IDEs and dev tools.
📡
Registry Persistence Monitor
14 Windows Registry keys used by malware families monitored via RegNotifyChangeKeyValue() — a kernel API that uses zero CPU when idle. New Run keys, IFEO hijacks, AppInit DLLs, and Winlogon changes trigger CRITICAL alerts instantly.
🌐
Cloud C2 Command & Control
Multi-tenant cloud server on Railway + PostgreSQL. Live fleet telemetry, remote host isolation from browser, per-organisation IT admin portal with scoped visibility. Seat enforcement, offline sweeper, role-based access control.
☁️
Cloud Disaster Recovery
Google Drive OAuth 2.0 backup. Per-machine encrypted cloud vaults identified by hardware machine_id. Two-tier restore: local vault → cloud rescue. Folder structure backup with selective extension and size filtering.
🧬
Threat Intelligence
SHA-256 hash of every new or modified file checked asynchronously against MalwareBazaar (abuse.ch). Results cached locally in SQLite for 7 days — zero network calls on repeat encounters. Optional VirusTotal integration via API key.
🚨
Dual-Channel Alerting
Discord/Slack rich embeds with color-coded severity, real-time mobile push. SMTP email alerts with AES-encrypted forensic .dat attachments for compliance trail. Forensic snapshots capture system state, disk info, process list, and last 15 log lines.
